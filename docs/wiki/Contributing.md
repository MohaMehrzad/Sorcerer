# Contributing

## Before Opening a PR
- Open or comment on an issue/discussion with context.
- Keep changes scoped and reversible.
- Include verification output.

## Required Checks
```bash
pnpm lint
pnpm exec tsc --noEmit
pnpm build:all
```

## Communication
- Ask questions in Discussions (`Q&A` category).
- Share ideas in Discussions (`Ideas` category).
- Follow the Code of Conduct.
