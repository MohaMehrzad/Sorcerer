# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-02-14

### Added
- Multi-agent and single-agent autonomous execution modes
- Checkpoint/resume run persistence and continuation packets
- Long-term memory retrieval, pin/forget, import/export
- Adaptive budget expansion for file writes and command runs
- Supervisor recovery cycles for failed/incomplete runs
- Completion contract validation for acceptance-quality finishing
- Reliability controls: preflight, strict verification, auto-fix loop
- Security controls: command allowlist and sensitive path protections
